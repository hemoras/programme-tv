# Base historique des chaînes de télévision francophones

## Objectif

Cette base recense les chaînes de télévision francophones ayant été distribuées en France (et les chaînes francophones de Belgique, Suisse, Luxembourg et Monaco).

Elle est historique : une chaîne disparue est conservée.

---

# Une ligne = une identité commerciale

Une nouvelle ligne est créée lorsqu'une chaîne adopte un nouveau nom commercial.

Exemples :

Direct 8 → D8 → C8

Fox Kids → Jetix → Disney XD

Télé Monte-Carlo → TMC

---

# Date de création

La date de création correspond :

- au début des émissions sous ce nom ;
- et non à la création de la société éditrice.

---

# Date de fin

La date de fin correspond :

- au dernier jour de diffusion sous ce nom ;
- vide si la chaîne existe encore.

---

# Prédecesseur

Nom commercial immédiatement précédent.

Exemple :

C8 → D8

France 2 → Antenne 2

RTL9 → RTL Télévision

---

# Groupe

Le champ contient tous les groupes successifs.

Exemple :

Bolloré Média (2005-2012), Groupe Canal+ (2012-)

ou

CLT (1995-1997), AB Groupe (1997-2017), Mediawan Thematics (2017-)

---

# Pays

Pays d'origine de la chaîne.

Valeurs actuellement admises :

- France
- Belgique
- Suisse
- Luxembourg
- Monaco

# Chaînes incluses

Sont incluses :

- chaînes nationales
- chaînes TNT
- chaînes hertziennes
- chaînes câble
- chaînes satellite
- chaînes ADSL/IPTV
- chaînes payantes
- chaînes gratuites
- chaînes disparues
- chaînes événementielles possédant une identité propre
- chaînes francophones de Belgique
- chaînes francophones de Suisse
- chaînes francophones du Luxembourg
- chaînes francophones de Monaco

---

# Chaînes exclues

Ne sont pas incluses :

- chaînes locales
- versions HD
- versions UHD
- versions 3D
- versions +1
- simples flux techniques
- canaux numérotés (Canal+ Live 1, Eurosport 360 4...)

---

# Renommages

Un changement de nom crée toujours une nouvelle ligne.

Même si :

- le logo évolue peu
- l'habillage change peu
- le programme reste identique

---

# Doublons

Le nom commercial ne peut apparaître qu'une seule fois.
